# BuffBuilder

> A Unity gameplay prototype inspired by Balatro, focusing on buff building and resource progression.

## Project Overview

BuffBuilder is a gameplay prototype developed with Unity 2022.3 LTS.

The project focuses on exploring:
- Buff System
- Resource Progression
- Gameplay Decision Making
- Data-driven Design

Instead of reproducing Balatro, this project aims to learn how roguelike buff systems and economic systems are designed and implemented.

## Tech Stack

- Unity 2022.3 LTS
- C#
- TextMeshPro
- Git
- GitHub
